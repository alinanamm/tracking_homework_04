## План по ДЗ MLOPS
1. Обучить модель
  1. Датасеты Train_1 и Test_1 выдаются изначально.
  2. Датасеты Train_2 и Test_2 выдаются для построения дообучения.
  3. Используя python и библиотеку pytorch на основе предоставленного шаблона в jupyter notebook обучить модель на предоставленном датасете Train_1. На основе датасета Test_1 провести оценку качества обученной модели по шаблону из jupyter notebook
2. Интегрироваться с tensorboard
  1. Перед стартом обучения сохранить параметры обучения. В процессе обучения, в конце итерации эпохи, залогировать метрики на основе которых будет оцениваться качество обучения
  2. На этапе тестирования модели залогировать метрики оценки качества обученной модели
3. Интегрироваться с s3 
  1. Используя docker-compose команду из шаблона jupyter notebook развернуть s3 minio локально.
  2. Выложить обученный артефакт модели после завершения обучения в s3.
4. Построить DVC пайплайн
  1. На основе датасета Train_2 дообучить полученную модель в из шага 1. Если используется s3, то модель перед дообучением должна скачиваться из s3.
  2. На основе датасета Test_2 провести тестирование дообученной модели.
  3. Выложить новую версию модели в s3.
  4. Перед стартом дообучения сохранить параметры дообучения. В процессе дообучения, в конце итерации эпохи, залогировать метрики на основе которых будет оцениваться качество дообучения
  5. Создать dvc пайплайн который автоматически воспроизводит шаги 1-4 из пункта 4. Распечатать полученный граф
5. Сравнить 2 версии модели по метрикам обучения и тестовым метрикам и написать вывод о полученных результатах.
6. Итоговый jupyter notebook с выполненными ячейками и выводом выложить в git.
